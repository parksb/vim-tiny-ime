vim-tiny-ime
========
Tiny automatic IME switcher for macOS. Switch to English IME whenever vim switchs
to normal mode.

```vim
" ~/.vimrc
Plug 'simnalamburt/vim-tiny-ime', { 'do' : './build' }
```
